Mis Proyectos

Proyecto vehiculos-caamano

Se trata de una página responsiva de una concesionaria de autos usados, es ficticia es decir que dichos autos y concesionarios no existen. Para el proyecto use html, css, scss, git, github, bootstrap y vercel. Cuenta con 5 html que son navegables y 6 html que se encuentran dentro de financiación. Utilice bootstrap para las cartas, el nav y el carrousel, con Scss cree estilos.scss y ahí importe todos los parciales. Use git para versionar el codigo, tambien cree una rama1 donde la usaba para trabajar, si el código funcionaba bien, lo mergeaba a la rama master y de ahí lo subí a github. El footer cuenta con el logo de twitter y facebook que son clickeables y te abren una ventana nueva con dicha red social. Para el proyecto utilice prácticas como usar el ampersand, camelCase, tabulación, etiquetas semánticas, metodología beem etc. Aplique una mejora al SEO optimizando las imágenes en Tinyjpg, respetando el uso de h1 h2 h3 y meta datos de keywords y content.
